# Contributing to Wordle Solver

Thank you for your interest in contributing!

## Reporting Issues or Bugs

Please use the issue templates provided when creating a new issue.

## Submitting Pull Requests

1. Open an issue first to discuss your proposed changes
2. Fork the repository and create a feature branch
3. Make your changes
4. Ensure the following pass:
   ```bash
   cargo fmt
   cargo clippy
   cargo test
   ```
5. Submit a PR using the pull request template

## Questions?

Feel free to open an issue or reach out to the maintainers.

Thank you for contributing!
